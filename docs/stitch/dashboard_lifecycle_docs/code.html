FastAPI Integrations

DeepContext provides a high-performance REST interface for direct engine manipulation. All endpoints are secured via Bearer tokens and return JSON-LD compliant objects.

GET /health

Returns current system metrics, CPU load, and cluster availability.

# Response Example
{
  "status": "operational",
  "engine_v": "2.4.1",
  "metrics": {
    "cpu": 0.428,
    "mem_gb": 14.2
  }
}
POST /add

Injects new data fragments into the extraction pipeline for immediate processing.

  "source": "whitepaper_v2.pdf",
  "priority": "high",
